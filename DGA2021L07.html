Lecture 07

Metrics

A scalar product on $T_pM$ is a bilinear symmetric positive definite form, i.e. a function from $T_pM\times T_pM$ to $\mathbb{R}$ which is bilinear, symmetric and positive definite.

The space of linear functions from $T_pM$ to $\R$ is denoted by $T_p^*M$ (the cotangent space at $p$); given $\alpha_p,\beta_p\in T_p^*M$, we define $\alpha_p\otimes\beta_p:T_pM\times T_pM\to\R$ by $$\alpha_p\otimes\beta_p(X_p,Y_p)=\alpha_p(X_p)\beta_p(Y_p)\;.$$

This is a bilinear form; however it will not be symmetric unless $\alpha_p=\beta_p$. However, we can take linear combinations, so we define $\alpha_p\beta_p=(\alpha_p\otimes\beta_p+\beta_p\otimes\alpha_p)/2$ i.e. $$\alpha_p\beta_p(X_p,Y_p)=\frac{1}{2}(\alpha_p(X_p)\beta_p(Y_p)+\alpha_p(Y_p)\beta_p(X_p))\;.$$

In general, a bilinear form on $T_pM$ is a combination of terms $\alpha_p\otimes\beta_p$ and a symmetric bilinear form is a combination of terms $\alpha_p\beta_p$.

Riemannian metric

So, a scalar product is a particular element of $T_p^*M\otimes T_p^*M$ (the set of bilinear forms) which is symmetric and positive definite.

As we defined the tangent bundle, we set $$T^*M\otimes T^*M=\{(p,\mathfrak{b})\ :\ p\in M,\ \mathfrak{b}\in T_p^*M\otimes T_p^*M\}\;.$$

A Riemannian metric on $M$ is a smooth section $g:M\to T^*M\otimes T^*M$ such that, for any $p\in M$ and $X_p,\ Y_p\in T_pM$, we have $g_p(X_p,Y_p)=g_p(Y_p,X_p)$ and $g_p(X_p,X_p)>0$ for all $X_p\in T_pM\setminus\{0\}$.
In local coordinates

Given a coordinate chart $(U,\phi)$ of $M$, where we denote the coordinates by $x^1,\ldots, x^k$, we have the local frame for $TM$ on $U$ $\partial_1,\ldots, \partial_k$. The local coframe given by the coordinates is $\{dx^1,\ldots, dx^k\}$ where $$dx^j(\partial_i)=\partial_i(x^j)=\delta_i^j\;.$$

Therefore, we have the sections $dx^i\otimes dx^j$ which generate $T^*M\otimes T^*M$ on $U$ and the sections $dx^idx^j$ which give at each $p\in U$ a basis of the subspace of symmetric bilinear forms.

With respect to these basis, a metric is usually given as $$g=g_{ij}dx^i\otimes dx^j=g_{ij}dx^idx^j\;,$$ where $g_{ij}=g_{ji}$. With this notations, $g(X,Y)=g_{ij}X^iY^j$.

Metric structure

Let $M$ be a $k$-dimensional manifold and $g$ a Riemannian metric on $TM$. The pair $(M,g)$ is called a Riemannian manifold.

Given a (piecewise) smooth curve $\gamma:[a,b]\to M$, its length is $$L_g(\gamma)=\int_a^b|\dot{\gamma}(t)|_gdt$$ where $|\cdot|_g$ is the norm induced by $g$ on each $T_pM$.

If $s:[a',b']\to [a,b]$ is any diffeomorphisms, then $\tilde{\gamma}=\gamma\circ s$ is a reparametrization of $\gamma$. We have $$L_g(\gamma)=L_g(\tilde{\gamma})\;.$$

Riemannian distance

Given a Riemannian manifold $(M,g)$, we define $$\mathrm{dist}_g(p,q)=\inf\{L_g(\gamma)\ :\ \gamma:[0,1]\to M,\ \gamma(0)=p,\ \gamma(1)=q\}$$ and we call it the Riemannian distance between $p$ and $q$. This is a distance, i.e.

  1. $\mathrm{dist}_g(p,p)=0$ $\forall\,p\in M$
  2. $\mathrm{dist}_q(p,q)=\mathrm{dist}_g(q,p)$ $\forall\,p,q\in M$
  3. $\mathrm{dist}_q(p,q)+\mathrm{dist}_g(q,r)\geq\mathrm{dist}_g(p,r)$ $\forall\,p,q,r\in M$.
Local isometries

If $(M,g)$ and $(N,h)$ are Riemannian manifolds, $F:M\to N$ is called a local isometry if the map $$dF_p:T_pM\to T_{F(p)}N$$ is an isometry between the Euclidean spaces $(T_pM, g_p)$ and $(T_{F(p)}N, h_{F(p)})$

If the map $F$ is also a diffeomorphism, then it is called a global isometry (and it is an isometry between $(M,\mathrm{dist}_g)$ and $(N,\mathrm{dist}_h)$).

In general $F^*h$ is the metric on $M$ defined by $$(F^*h)_p(X_p,Y_p)=h_{F(p)}(F_*X_p, F_*Y_p)$$ so $F$ is a local isometry if $F^*h=g$.

Embedded submanifolds

If $i:M\to \overline{M}$ is an immersion and $i(M)$ is a submanifold of $\overline{M}$, then $i$ is called an embedding. Suppose that $\overline{g}$ is a Riemannian metric on $\overline{M}$.

The metric $g=i^*\overline{g}$ on $M$ is called induced metric and $(M,g)$ is a Riemannian submanifold of $\overline{M}$.

This is equivalent to defining the metric on the tangent space of the submanifold by restricting the metric of the ambient space.

If $\gamma:I\to M$ is a curve, $L_g(\gamma)=L_{\overline{g}}(\gamma)$, but for $p,q\in M$, $\mathrm{dist}_g(p,q)\geq \mathrm{dist}_{\overline{g}}(p,q)$.

Musical isomorphism

Given a vector-field $X\in\mathfrak{X}(M)$ and a Riemannian metric $g$ on $M$, we define the $1$-form $X^\flat\in\Omega^1(M)$ by $$X^\flat(Y)=g(X,Y)\qquad \forall\,Y\in\mathfrak{X}(M)\;.$$

In coordinates, $X^\flat=X^ig_{ij}dx^j$. This is called lowering the index or contraction of $g$ with $X$.

So, we have a map $\phantom{A}^\flat:TM\to T^*M$ which is an isomorphism of bundles (because $g$ is positive definite).

The dual (or inverse) metric

We consider, on $T^*_pM$, the metric such that, for any $X_p, Y_p\in T_pM$, we have $$g^{-1}(X^\flat_p,Y^\flat_p)=g(X_p,Y_p)$$

this is called dual (or inverse) metric. As, for a vector space, $V^{**}=V$, this metric on $T^*_pM$ gives a section of $TM\otimes TM$.

In coordinates, we write $g^{-1}=g^{ij}\partial_i\partial_j$, where $g^{ij}$ is the $(i,j)$ element of the matrix $(g_{ij})^{-1}$. That is, $$g^{ij}g_{jk}=\delta^i_k\;.$$

The sharp isomorphism

The inverse map of $X\mapsto X^\flat$ is denoted by $\phantom{A}^\sharp:T^*M\to TM$ and is induced by $g^{-1}$, which is again symmetric and positive definite.

Therefore $$\alpha^\sharp=\alpha_ig^{ij}\partial_j$$ (this is obviously called raising the index) and $(X^\flat)^\sharp=X$, because $g^{ij}g_{jk}=\delta^i_k$.

The gradient of a function

Let $(M,g)$ be a Riemannian manifold.

If $f:M\to\mathbb{R}$ is a smooth function, then the gradient of $f$ is the vector-field $$\mathrm{grad}f=(df)^\sharp$$ i.e. with respect to a local frame $$\mathrm{grad}f=g^{ij}(\partial_i f) \partial_j\;.$$
